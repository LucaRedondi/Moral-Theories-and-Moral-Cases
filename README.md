# Moral-Theories-and-Moral-Cases
In this dataset, we collect moral theories as well as moral cases. 
Moral cases are mostly extracted from M.J. Sandel "Justice: What is the right thing to do?", a book on political philosophy and contemporary ethics rich in examples. 
As for the moral theories, we present the foundational principles rather fully developed system of duties. Perhaps our descriptions are better understood as moral frameworks, that can be instantiated to create concrete normative theories. We cover several forms of Cosequentialism, Deontology and Virtue Ethics. The description are extracted from several entries of the Stanford Encyclopedia of Philosophy.
